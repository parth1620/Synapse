# Synapse
 ## Synapse Unsupervised Models
 Requirements
```console
$ pip install numpy
$ pip install pandas
$ pip install plotly
$ pip install streamlit
```

```console
$ streamlit run https://github.com/parth1620/Synapse/blob/master/Synapse/Synapse%20Unsupervised%20Model.py
```
