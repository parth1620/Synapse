# Synapse
 ## Synapse Unsupervised Models
 - K-means Clustering
 - Principal Component Analysis
 Requirements
```console
$ pip install numpy
$ pip install pandas
$ pip install plotly
$ pip install streamlit
```
In terminal run
```console
$ streamlit run https://github.com/parth1620/Synapse/blob/master/Synapse/Synapse%20Unsupervised%20Model.py
```
